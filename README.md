# AgroManager Web (Frontend)

Dashboard frontend para AgroManager. Proyecto independiente del backend.

**Ruta del proyecto:** `C:\Users\Usuario\source\repos\AgroManagerWeb`

## Stack

- Next.js, React, Tailwind CSS, TypeScript

## Cómo ejecutar ejemplo

```bash
cd C:\Users\Usuario\source\repos\AgroManagerWeb
npm install
npm run dev
```

Abre http://localhost:3000. Login con cualquier email/contraseña (simulado). Para conectar con tu backend más adelante, configura la URL del API y sustituye los mocks en `src/data/mock.ts` por llamadas `fetch`.
